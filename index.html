HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The P element represents a new paragraph or block of text.
  • Stands for Hypertext Markup Language
  • HTML gives the website Structure also known to be a frame for our work.
  • Elements are usually represented by angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules or declaration will apply.
  • Declarations contain two important components, the CSS property we want to apply and the value of the property.

Git

  • Free, open source version control system.
  • Allows web developers to collaborate on projects without overwriting each other's work.
  • Allows developers to revert to previous versions of the code if a problem is discovered
  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it.
  • git checkout main: Switches to main branch.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An operator is a math symbol that produces a result based in two values.
  • Conditionals are structures used to test if an expression returns true or false (If,else).
  • Functions are a way of packaging funcionality that you wish to reuse.
  • Functions may look like a variable but is followed by parenthesis.
  • Arguments go inside the parenthesis, seperated by commas.
  • Events are code structures that listen to the activity in the browser, and run code in response. An exmple would be the "Click Event" (Mouse Clicking).
  • An array is a single variable that is used to hold a group of data.
  • Arrays are especially useful when we want to perform a task on multiple pieces of work at once.
  • Arrays can store different types of data, such as strings, numbers and so on.
  • Count zeros ex: 0,1,2,3... not 1,2,3...
  • Commenting out code makes it inactive without deleting it.
